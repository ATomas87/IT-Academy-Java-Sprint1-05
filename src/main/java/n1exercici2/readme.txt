El programa ya está compilado, por lo que para poder usar este programa sólo hay que dirigirse a la carpeta src/main/java en el terminal y,
desde allí, ejecutar el comando "java n1exercici2.ListDirectoryRecursive <Ruta del directorio>".

Hay que tener en cuenta que la variable de entorno PATH debe incluir los binarios Java. En caso de no ser así hay que ejecutar el programa
introduciendo toda la ruta del binario Java junto con la ruta del archivo niexercici2.ListDirectoryRecursive.
